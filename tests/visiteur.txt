== Filtrer la page d'accueil ==

ÉTANT DONNÉ QUE: je suis visiteur 
QUAND: je filtre les bloc par  la date
		ET: aucun bloc existe
ALORS : je suis redirigé vers la page Accueil
		ET: le message d’erreur « Aucun résultat trouvé » est affiché

ÉTANT DONNÉ QUE: je suis visiteur
QUAND: je filtre les bloc par  la date
		ET: un bloc existe 
ALORS : je suis redirigé vers la page Accueil
		ET: un résultat est retourné

ÉTANT DONNÉ QUE: je suis visiteur

ALORS : je suis redirigé vers la page Accueil
		ET: le message d’erreur « Aucun résultat trouvé » est affiché

ÉTANT DONNÉ QUE: je suis visiteur 
QUAND: je filtre les bloc par titre
		ET: le titre existe
ALORS: je suis redirigé vers la page Accueil
		ET: un résultat est retourné

ÉTANT DONNÉ QUE: je suis visiteur 
QUAND: je filtre les bloc par media
		ET: le media est inexistant
ALORS: je suis redirigé vers la page Accueil
		ET: le message d’erreur « Aucun résultat trouvé » est affiché

ÉTANT DONNÉ QUE: je suis visiteur

== Connexion ==


ÉTANT DONNÉ QUE: je suis visiteur
QUAND: je clique sur se connecter
ALORS: une modal box est affichée

ÉTANT DONNÉ QUE: je suis visiteur
QUAND: je rempli les champs de la modal box connexion 
		ET: je valide
		ET: mon mot de passe est valide
		ET: mon pseudo est valide
ALORS: je suis redirigé vers la page Accueil
		ET: un lien logout est affiché sur la nav bar

ÉTANT DONNÉ QUE: je suis visiteur
QUAND: je rempli les champs de la modal box connexion
		ET: je valide
		ET: mon mot de passe est pas valide
		ET: mon pseudo est pas valide
Alors le message d’erreur « passe et ou pseudo non valides » est affiché
ET la modal box reste affichée

ÉTANT DONNÉ QUE: je suis visiteur
QUAND: je rempli les champs de la modal box connexion 
		ET: je valide
		ET: mon mot de passe est valide
		ET: mon email est valide
ALORS: je suis redirigé vers la page Accueil
		ET: un lien logout est affiché sur la nav bar

ÉTANT DONNÉ QUE: je suis visiteur
QUAND: je rempli les champs de la modal box connexion
		ET: je valide
		ET: mon mot de passe est pas valide
		ET: mon email est pas valide
Alors le message d’erreur « passe et ou pseudo non valides » est affiché
ET la modal box reste affichée

ÉTANT DONNÉ QUE: je suis visiteur
QUAND: je rempli les champs de la modal box connexion
		ET: je valide
		ET: mon mot de passe est pas valide
		ET: mon email est pas valide
		ET: mon pseudo est pas valide
		ET: le nombre des essais est égal a 3
ALORS:  je suis redirigé vers la page Accueil
		ET: le message d’erreur « la connexion au site est bloquée pendant 15mn» est affiché
		ET: le lien de connexion est non actif
